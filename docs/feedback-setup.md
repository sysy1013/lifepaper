# 건의사항 이메일 전송 설정

**💬 건의사항** 탭에서 보낸 내용은 앱 어디에도 저장되지 않고 운영자 메일함으로 바로 전송됩니다.
전송이 동작하려면 운영자가 배포 환경(Streamlit Cloud → App settings → **Secrets**)에
아래 값을 등록해야 합니다. 로컬 개발이라면 `.streamlit/secrets.toml`에 같은 내용을 넣습니다.

```toml
FEEDBACK_TO = "받는사람@example.com"
SMTP_HOST = "smtp.gmail.com"
SMTP_PORT = 465
SMTP_USER = "보내는계정@gmail.com"
SMTP_PASSWORD = "앱 비밀번호 16자리"
```

- `SMTP_PORT`는 생략할 수 있으며, 생략 시 465(SSL)로 동작합니다.
- Gmail을 쓸 경우 `SMTP_PASSWORD`에는 **앱 비밀번호**(Google 계정 → 보안 → 2단계 인증 →
  앱 비밀번호에서 발급하는 16자리)를 넣습니다. **일반 로그인 비밀번호는 동작하지 않습니다.**
- 실제 비밀번호·수신 주소는 저장소(Git)에 **절대 커밋하지 마세요.** Secrets에만 넣습니다.

## 설정하지 않으면?

건의사항 탭은 그대로 동작합니다. 다만 전송 대신
"현재 이메일 전송이 설정되어 있지 않습니다"라는 안내와 함께 작성한 내용이
복사 가능한 형태로 표시되어, 교사가 직접 복사해 전달할 수 있습니다.
작성 내용이 사라지지는 않습니다.
